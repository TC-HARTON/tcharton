Press Release ／ 機械検証データ公開

静岡県 1,553 社の WEB 品質を機械検証 — Stella データを CC BY 4.0 で公開

2026 年 5 月 15 日 10:00 JST 公開 / T.C.HARTON / 沼津市発

プレスリリース要旨

沼津市の T.C.HARTON（代表 大内 達也）は、2026 年 4 月から実施した静岡県内 11 業種 1,553 社の WEB サイト品質を 4 軸（パフォーマンス / SEO / アクセシビリティ / セキュリティ）で機械検証した結果を、Stella サブセクション内で Creative Commons Attribution 4.0 International（CC BY 4.0）ライセンスとして公開しました。地域の中小企業が自社サイトの位置づけを把握できる、独立第三者による集計データです。

＜本文＞

WEB 制作・AI 予測サービスを提供する T.C.HARTON（所在地：静岡県沼津市大岡 2690、代表：大内 達也）は、2026 年 5 月 15 日、独自に開発した機械検証エンジン「Stella Scanner」を用いて、静岡県内の 11 業種 1,553 社の WEB サイト品質を 4 軸で評価した結果を、Stella サブセクション内で公開しました。

本データは CC BY 4.0 ライセンスのもとで自由に利用可能で、出典を明記すれば商用・非商用を問わず再利用・引用・再配布できます。個別企業名は非公開とし、業種別の中央値・上位値・致命的 NG 比率といった集計データのみを提供します。

公開データの主な内容

・対象：静岡県内 11 業種 1,553 社の WEB サイト

・評価軸：パフォーマンス（Core Web Vitals）/ SEO / アクセシビリティ / セキュリティの 4 軸

・集計指標：業種別中央スコア / 業種別上位値 / 致命的 NG 検出率 / ★ 認定取得率

・公開形式：HTML 集計表（Stella サブセクション）+ 機械可読 Dataset（JSON-LD）

・ライセンス：CC BY 4.0（出典明記で自由利用可）

・個別企業名：非公開（公正性担保のため）

公開の背景

地方の中小企業の多くは、自社の WEB サイトが業界平均と比べてどの位置にあるかを客観的に把握できる手段を持っていません。「制作してそのまま」になりがちな構造を、独立第三者による機械検証データで可視化することが本データ公開の目的です。

WCAG 2.2 AA（2023 年 10 月勧告）、Verizon 2025 DBIR（中小企業ランサム被害 88%）、Patchstack 2025 年版（WordPress エコシステム脆弱性 11,334 件中 91% プラグイン由来）等、2025 年に公表された WEB 品質関連の最新基準を機械検証ロジックに取り込んでおり、業界の現状把握に有用なベンチマークとして提供します。

代表コメント

「自社サイトの品質を客観的に把握できないまま、制作会社に言われるがまま費用を払い続けている地方の事業者を、何度も見てきました。Stella は誰でも自由に参照できる集計データを軸に、地方の WEB 品質の底上げを目的にしています。今回のデータ公開は出発点であり、毎月の継続スキャンで時系列の品質改善も可視化していきます。」

— T.C.HARTON 代表 大内 達也

今後の展開

HARTON は今後、検証対象を月次で更新し、時系列での WEB 品質改善トレンドを公開する予定です。また、業種・地域を順次拡大し、2026 年内に全国主要都市まで対象を広げる計画です。本データを活用した報道・研究・分析等の引用は、CC BY 4.0 のライセンス条件（出典明記）に従い、自由に行うことができます。

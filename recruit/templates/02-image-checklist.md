# 02. 画像素材リスト (チェックシート)

> 提出予定枚数 合計目安: **10-30 枚**
> 形式: JPG / PNG / WebP / HEIC (スマホ撮影 OK)
> サイズ: 長辺 1920px 以上推奨 (大きい分には縮小可)

---

## §3 提出方法

### 推奨: ZIP 圧縮 + オンラインストレージ共有

```
all-images.zip
├── logo/
│   ├── logo.png
│   ├── logo-white-bg.png (任意)
│   └── logo-mark.png (任意)
├── service/
│   ├── service-01.jpg
│   ├── ...
│   └── service-10.jpg
├── staff/
│   ├── staff-01-rep.jpg
│   └── ...
├── office/
│   ├── office-01-exterior.jpg
│   └── ...
└── optional/
    ├── customer-01.jpg
    └── ...
```

ZIP を Google Drive / Dropbox / OneDrive に上げ、共有 URL を `contact@tcharton.com` にメール。

### 代替: メール添付 (10MB 以下の場合のみ)

合計 10MB 以下なら直接メール添付可。

## §4 画像権利の確認

応募時に以下を承諾いただきます:

- [ ] 提出画像は応募者または応募者の法人が**著作権を保有**している
- [ ] 第三者 (顧客 / モデル等) が写る画像は **本人掲載許可済**
- [ ] T.C.HARTON が WEB サイト構築 / 事例公開 (tcharton.com / note) で使用する事を許諾する
- [ ] 解像度・品質はそのまま使用可 (大幅な加工が必要な場合は事前協議)

導入事例 — ★ 取得への道筋

2026-05-03 公開 ／ 最終更新 2026-05-03

HARTON Stella は機械検証で WEB 品質を測る独立認定機関である。本ページは ★ 認定を取得した事業者が辿った「課題 → 改善 → 取得」の 3 段ストーリーを公開する。Phase 0.5 静岡県 5 都市 902 件機械検証では ★ 認定取得は 0 件であるが、運営主体 T.C.HARTON の自社サイト tcharton.com が ★★★ HARTON Stella S-Class を取得した自己実証体 第 1 号として、全事業者に向けて取得経緯を verbatim 公開する。

「機械検証が示す、AI 時代の WEB 品質。」 — HARTON Stella

自己実証体 第 1 号

  • ★★★ HARTON S-Class

    tcharton.com — 沼津起点の自己実証 (旧 C/65 → 🏆 S/90)

    IT・WEB 制作 ／ 静岡県沼津市 ／ 取得 2026-04-27 / commit 36d4328

    運営主体 T.C.HARTON の自社サイト。dogfooding 倫理に基づき、自社評価基準を自社サイトに先に適用した第 1 号。旧スコア C/65 から 7 commit と 1 個の住所完全公開で 🏆 S/90 (HARTON S-Class) を取得した記録的事例。

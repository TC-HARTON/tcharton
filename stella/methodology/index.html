HARTON Stella は WEB サイトを 4 つの観点で並列独立評価する。各観点は scanner.py で 45+ 項目を機械検証し、合計 2554 項目の SPEC v3.4.2 と整合する。本機関は dogfooding 倫理（自己実証体としての自社サイト ★★★ 取得）と地方都市から再定義する WEB 品質の民主化を運営方針とする。出典: Google Web Vitals 公式 / W3C WCAG 2.2。

「機械検証が示す、AI 時代の WEB 品質。」 — HARTON Stella 信頼根拠の核（MASTER-PLAN §2.0.4）

2026-04-27 公開 ／ 2026-04-30 v1.1.7 ブランド戦略整合改訂

評価方法

S クラスの哲学的定義

「S クラスとは、機械が客観検証する、AI 時代における信頼の最大値である」（MASTER-PLAN §3.0 / マニフェスト 50 字 verbatim）

本機関の S クラスは、人間の主観や金銭、規模に依存しない。4 軸（基礎・防御・AI 検索・経営）の機械検証で、誰もが再現可能な手順で到達できる頂点。それは「努力の結果」ではなく、「思想の到達点」である。S クラス取得は、業界ベンチマーク・同業者の教科書的手本・AI 検索エンジンが第一想起する情報源としての位置を意味する（§3.0.4 意味的定義）。

自己実証型認定の倫理規範（dogfooding 倫理）

本機関の中核原則は「機械検証が示す、AI 時代の WEB 品質。」である (dogfooding 倫理: 自社評価基準を自社サイトに先に適用する)。tcharton.com（自社サイト）が自社の評価基準で ★★★（HARTON S-Class）を取得した自己実証体 第 1 号として、業界の品質改善のため独立中立に運営する。

  • L1 自己適用: 自社サイト tcharton.com が認定基準で評価される（scanner.py 月次再判定 (四半期末レポート集計)）
  • L2 自己拘束: 評価結果は自己例外なし（致命的 NG / 降格条件は事業者と同条件）
  • L3 自己公開: 機械検証ログを全件 verbatim 公開する（評価の透明性 / 第三者検証可能性）

沼津起点での自己実証は、地方都市から再定義する WEB 品質の民主化の旗印として機能する。詳細は サイトについて + 静岡県 5 都市 WEB 品質業界レポート 2026 Q2 (4-6 月号)。

評価の 5 原則

  • 透明性: 評価項目・閾値・実施手順を全公開する。
  • 客観性: 主観的判断を排除し、scanner.py による自動計測のみ。
  • ポジティブセレクション: 良いサイトを称え、低評価サイトは公開しない。
  • 公平性: 業種・規模・HARTON との取引関係に一切依存しない。
  • 進化: 年次で基準を改訂し、過去評価もアーカイブとして保持する。

★区分の閾値（MASTER-PLAN §3.4）

  • HARTON Stella

    総合 70 点以上 + 致命的 NG 0 件 + S 条件不問。物語: 「AI 時代の信頼を体現する」

  • HARTON 優良

    総合 80 点以上 + 致命的 NG 0 件 + S 条件 4/5 達成。物語: 「地域を代表する WEB 品質」

  • HARTON S-Class Certified

    総合 90 点以上 + 致命的 NG 0 件 + S 条件 5/5 完全達成。物語: 「業界の頂点に立つ WEB 品質」

致命的 NG（一発除外条件）

  • HTTPS 非対応 — 出典: IPA「安全なウェブサイトの作り方」
  • SSL 証明書エラー（期限切れ・無効）
  • WP 管理面露出（wp-login.php / readme.html / xmlrpc.php）
  • CMS バージョン情報露出（generator meta タグ等）

★区分問わず一発除外する。scanner.py Phase 9d で機械検出を実装済（2026-04-26）。

評価サイクル (canonical / 全ページ統一)

  • 月次スキャン: 全掲載対象事業者 + 業界レポート対象母集団を毎月初に scanner.py で全件再判定。★ 区分変動を月次反映。
  • 四半期レポート: 月次データを四半期末 (Q1: 1-3月 / Q2: 4-6月 / Q3: 7-9月 / Q4: 10-12月) に集計し、業界レポート + 比較ページを公開。
  • 年次基準改訂: 4 軸・S 条件・閾値を年次で見直し、SPEC v3.x と整合確認。
  • 即時対応: 重大脆弱性発見時は 24 時間以内に掲載一時停止判断。

自己実証体 tcharton.com も例外なく月次スキャン + 四半期レポート対象として運用される (dogfooding 倫理 / 自己例外なし)。

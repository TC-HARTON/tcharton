司法書士 の WEB 品質評価ポイント

  • Trusted Types CSP 適用（個人情報窃取リスク防止 / 致命的 NG 直結）
  • 相続・登記相談フォームの SSL + form-action self CSP 制御
  • JSON-LD LegalService + ProfessionalService の重ね合わせ実装
  • 司法書士法 (e-Gov) 引用 + 公的リンク 5+ 件 (HSCEL §G-2)
  • 国家資格者 (司法書士登録番号) の Schema.org Person + hasOccupation 記述

司法書士 WEB 品質チェックリスト

  1. HTTPS + HSTS preload 登録済 / SSL 期限切れなしか
  2. 相談フォームが Trusted Types + CSP form-action で保護されているか
  3. JSON-LD で LegalService + ProfessionalService が両方記述されているか
  4. 司法書士登録番号 + 所属司法書士会が HTML で機械可読か
  5. <blockquote cite> で司法書士法 (e-Gov) を引用しているか
  6. 相続登記 / 不動産登記 / 商業登記 等のサービスが Schema.org Service で構造化されているか
  7. GBP 連携 + areaServed 記述で地域 SEO 強化されているか
  8. WCAG 2.2 AA: フォーム input に label for 関連付け済か

Phase 1 東京都 渋谷区 で信頼できる 司法書士 事務所のサイトを見分けるには？

HARTON Stella は WEB 品質を 4 軸機械検証で客観評価し、LegalService Schema.org 適合 + HTTPS + Trusted Types CSP の 3 条件を必須とする。Phase 1 東京都 渋谷区 の認定状況は本ページ上部で確認可能。

Phase 0.5 静岡県 5 都市の 司法書士 サイトを 18 件機械検証 (5 都市集計) し、★ 認定 0 件 / 業界最高点 54 点 / 致命的 NG 44.9% (2026-05-02 時点)。詳細は静岡県 WEB 品質業界レポート 2026 Q2 (4-6 月号)に公開。

司法書士 サイトで致命的 NG となる項目は？

HTTPS 非対応 / SSL 証明書無効 / 個人情報フォーム CSP 未設定 / CMS バージョン情報露出 の 4 項目が一発除外条件。司法書士は個人情報・登記情報を扱うため特に致命的。

司法書士 の Schema.org 推奨タイプは？

LegalService + ProfessionalService の重ね合わせを推奨する。サービスとして「相続登記」「不動産登記」「商業登記」「成年後見」を Service 子要素で構造化することで AI 検索引用率が向上する。

司法書士 事務所サイトが ★★★ を取得するための最重要改善は？

(1) HTTPS + HSTS preload 登録 (2) JSON-LD LegalService 完全実装 + 司法書士登録番号機械可読化 (3) Trusted Types CSP の順で改善する。

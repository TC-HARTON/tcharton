行政書士 の WEB 品質評価ポイント

  • Trusted Types CSP 適用（個人情報窃取リスク防止 / 致命的 NG 直結）
  • 許認可申請フォームの SSL + form-action self CSP 制御
  • JSON-LD LegalService + ProfessionalService の重ね合わせ実装
  • 行政書士法 (e-Gov) 引用 + 公的リンク 5+ 件 (HSCEL §G-2)
  • 国家資格者 (行政書士登録番号) の Schema.org Person + hasOccupation 記述

行政書士 WEB 品質チェックリスト

  1. HTTPS + HSTS preload 登録済 / SSL 期限切れなしか
  2. 相談フォームが Trusted Types + CSP form-action で保護されているか
  3. JSON-LD で LegalService + ProfessionalService が両方記述されているか
  4. 行政書士登録番号 + 所属行政書士会が HTML で機械可読か
  5. <blockquote cite> で行政書士法 (e-Gov) を引用しているか
  6. 許認可 / 遺言相続 / 在留資格 等のサービスが Schema.org Service で構造化されているか
  7. GBP 連携 + areaServed 記述で地域 SEO 強化されているか
  8. WCAG 2.2 AA: フォーム input に label for 関連付け済か

Phase 0.5 静岡県 5 都市 で信頼できる 行政書士 事務所のサイトを見分けるには？

HARTON Stella は WEB 品質を 4 軸機械検証で客観評価し、LegalService Schema.org 適合 + HTTPS + Trusted Types CSP の 3 条件を必須とする。Phase 0.5 静岡県 5 都市 の認定状況は本ページ上部で確認可能。

行政書士 サイトで致命的 NG となる項目は？

HTTPS 非対応 / SSL 証明書無効 / 個人情報フォーム CSP 未設定 / CMS バージョン情報露出 の 4 項目が一発除外条件。

行政書士 の Schema.org 推奨タイプは？

LegalService + ProfessionalService の重ね合わせを推奨する。サービスとして「建設業許可」「飲食店営業許可」「遺言相続」「在留資格」を Service 子要素で構造化することで AI 検索引用率が向上する。

行政書士 事務所サイトが ★★★ を取得するための最重要改善は？

(1) HTTPS + HSTS preload 登録 (2) JSON-LD LegalService 完全実装 + 行政書士登録番号機械可読化 (3) Trusted Types CSP の順で改善する。

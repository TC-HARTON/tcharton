よくある質問

浜松市 の 美容院 で WEB 品質が高い店は？

HARTON Stella は 4 軸機械検証で 美容院 サイトを評価する。浜松市 の認定状況は本ページ上部の獲得率で確認可能（現在 0/98 件 / 2026-05-02 時点）。

美容院 サイトで SNS 連携と Schema.org を両立するには？

Schema.org BeautySalon の sameAs 配列に Instagram / X / Facebook の公式 URL を記述する。CSP frame-src で SNS 埋込を許可し、Trusted Types policy で外部 JS の sink を制御する。

BeautySalon の Schema.org 最低限の実装は？

name + address + telephone + url の 4 項目が必須。さらに HairSalon を @type 配列に追加し、施術メニューを Service 子要素で構造化することで AI 検索引用率が向上する。

美容院 サイトの予約フォームと CSP を両立するには？

form-action 'self' https://api.web3forms.com 等の信頼バックエンドを CSP に明示し、Trusted Types policy で sink を制御する。外部予約 (ホットペッパー等) はリンクで連携し、iframe 埋込は frame-src 整合を確認する。

美容院 サイトが ★ 認定を取得できない最多原因は？

(1) HTTP 非 SSL (2) 予約フォーム外部 JS で CSP 違反 (3) WordPress 管理面露出 が 3 大要因。改善ガイダンスで詳細を公開している。

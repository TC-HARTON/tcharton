飲食店（レストラン・カフェ・パン屋）の要点

飲食店の WEB サイトは「食べたい」と感じる写真品質 + Google マップから 1 タップで予約が来店誘導の核です。観光地ではインバウンド需要を取り込む多言語対応も必須。HARTON はGoogle ビジネスプロフィール連携 + 予約導線 + 多言語 + 構造化データ（Restaurant / Menu）を標準実装します。

1. 飲食店サイトでよくある悩み

Google ビジネスプロフィールが「自動生成のまま」で、写真・営業時間・メニュー・電話が更新されていない。

予約フォームが「お電話のみ」で、夜間・休日のスマホ予約を取りこぼしている。

メニュー写真が暗い・小さい・古い。食欲を刺激しない品質で、Lindgaard の 50ms 第一印象テストで離脱される。

2. 必須実装項目（5 点）

  1. 1

    Google ビジネスプロフィール連携：写真 / 営業時間 / メニュー / 電話番号の正確な反映 + WEB サイトとの整合

  2. 2

    予約システム連携：TableCheck / OpenTable / 自社フォームのいずれかで 24h 予約対応

  3. 3

    写真の品質：プロカメラマン or スマホでも構図 + 光 + 編集を適切に。WebP / AVIF + 遅延読み込み

  4. 4

    多言語対応：英語・中国語・韓国語の最低 3 言語（観光地・インバウンド需要のある立地）

  5. 5

    Restaurant + Menu Schema：構造化データで Google リッチカード対応

3. 法令上の論点

食品衛生法に基づくアレルギー表示の正確性、景表法に基づく「絶品」「最高」等の自己評価表現の禁止、産地・原材料表示の正確性が論点。テイクアウト・デリバリーは食品表示法の追加要件もあります。

よくある質問

Google ビジネスプロフィールの最適化だけで効果はありますか？
Google ビジネスプロフィールの正確な運用は、ローカル検索順位の最も基本的な要因です。ただし「Google マップだけ」では予約率・客単価の最適化に限界があります。自社サイトと連携し、メニュー詳細・予約フォーム・店舗ストーリーを併設することで CV が大きく伸びます。
英語メニューを作りたいのですが、翻訳はどうすれば？
AI 翻訳のレビューを必ず人手で行うことを推奨します。料理名は文化背景まで伝える必要があり、機械翻訳のみでは誤解を招く危険があります。HARTON では翻訳パートナー（沼津・伊豆エリアの英文ライター）と連携可能です。
Instagram の写真を WEB サイトに自動取り込みできますか？
可能ですが、Instagram の埋め込みウィジェットは表示速度を著しく落とすことがあります。代わりに、Instagram 投稿の写真を WebP に変換して自社サーバーから配信する方式を推奨します。

よくあるご質問

他社で制作したサイトでも依頼できますか？

はい。当社制作以外のサイトも対象です。HARTON Stella は認定機関として中立評価を実施します。改修依頼への営業義務はありませんので、レポートのみのご利用も歓迎します。

レポートはどのくらいの分量ですか？

PDF 30〜50 ページ程度（規模により変動）。総合判定 + 5 評価軸スコア + 致命的 NG 一覧 + 改修優先度ランキング + 公式一次ソース紐付け + 業界比較ベンチマーク を網羅します。

★★★ がついたら何ができますか？

HARTON Stella 認定書（PDF）をご活用いただけます。サイトのフッターや RFP / 見積書等で「HARTON Stella ★★★ 認定」を表示することで、独自基準（WHATWG / WCAG / OWASP / GEO 準拠）による品質評価を訴求できます。評価基準・判定ロジックは /methodology/ で完全公開しています（認定の維持には月次再判定 + 月額保守契約が必須です）。

★ や評価なしでもレポートはもらえますか？

はい。判定結果に関わらず PDF レポートは納品します。「無評価」の場合でも、現状の客観評価 + 改修優先度ランキングは含まれますので、改善方針の策定にご活用いただけます。

改修依頼をしないと損をしますか？

いいえ。レポート単発の利用が主用途です。営業義務はありません。改修ご希望の場合のみ ★★★ 化改修 または ★★★ S クラス保証パッケージ をご提案します。

## 2. ペルソナ別の再訪問所感

### 沼津 50 代飲食オーナー（WEB 知識ほぼゼロ）— /pricing

- ❌ 静的 HTML カード内「Core Web Vitals 全項目 緑」「AI 検索対応 強化」が**完全に意味不明**。50 代オーナーはここで離脱リスク
- ✅ 追加費用一覧（ドメイン実費 / SSL 込み / 写真撮影任意）は**素晴らしい透明性**。「言われていなかったお金がかかる」恐怖を完全に解消

- hero「あなたに合う WEB 制作プランは、2 つあります。」が**選択肢を 2 つに絞ってくれる安心感**
- メリット / デメリット / 向いている方の 3 軸構成が**意思決定フォーマットとして秀逸**
- ❌ 静的 HTML メリット内「AI 検索対応 強化（GEO 最適化）」の括弧書きが**逆効果**。「GEO」を見て不安になる
- ❌ 比較表の「LLMO（AI 検索対応）」も同様。本文で AI 検索と書きながら表でだけ LLMO 表記の**不整合**

### 三島 40 代美容室

- 「代表 大内 達也（データサイエンス 15 年）が直接設計」の明示は**人格による信頼**を生む

### 沼津 35 代製造業二代目

## 5. 結論

V2→V3 で **+6 点（82→88）**。3 つの改修のうち TOP 統一と llms-full.txt は **完全達成**だが、ジャーゴン削減は **置換漏れあり部分達成**。最大の機会損失は **pricing と services/web に残る 12 件のジャーゴン**で、これは静的置換で 1 時間以内に +2 点回収可能。続いて TOP 業種チップと顧客実名の声で +4 点、合計 **94 点が現実的な V4 目標**。

100 点満点には **methodology / faq / about のリライト + 動画コンテンツ（代表メッセージ 60 秒）** が必要だが、V4 で 94 点に達すれば、沼津・静岡圏の中小企業オーナー実流入では **競合比較で実質トップ品質**として成立する。

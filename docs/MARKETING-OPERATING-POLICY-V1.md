## 0. 結論（運用方針の核）

**「技術スコアを主役にしない。データサイエンス専門性 × AI 最適化を『成果の裏付け』として、静岡県東部にエリア限定で旗を立てる。」**

調査で判明した最重要事実:
- Lighthouse スコア単体は中小企業オーナーに刺さらない。「数値 → 集客・売上・問い合わせ件数」への翻訳が必須
- 価格 35 万円〜 は「テンプレ最安帯」と並ぶと埋もれる。「同額でも中身が違う」訴求が必要
- AIO/LLMO の先行者優位はまだ取れるが、専門会社の参入が急増中 → 2026 年内着手が最終局面
- 沼津の競合の構造的弱点 = ①価格非開示が多数 ②実績が「業種数」止まりで成果数値なし ③自社サイトの技術品質を証拠化していない

## 1. ポジショニング

| 項目 | 方針 |
|---|---|
| 一言 | 静岡県東部で唯一、AI 最適化を実装し品質を機械検証で証明する WEB 制作者 |
| 主訴求 | 「AI に、選ばれる WEB を。」（成果 = 見つかる・選ばれる） |
| 副訴求 | 機械検証で品質を証明する WEB 制作 + AI 予測（裏付け = 安心） |
| 避けるべき | 「Lighthouse 100 点」単独訴求 / 「格安・短納期」勝負 / 全国の LLMO 専門会社との真っ向勝負 |
| 競合に対する立ち位置 | 格安勢（ちょこWEB/ロコプリオ）とマーケ型（メディアデザイン/PC11）の「第三軸」 |

## 3. 集客チャネル運用方針（優先順）

| 優先 | チャネル | 具体運用 | 根拠 |
|---|---|---|---|
| **P0** | 紹介導線 | 既存接点・recruit キャンペーン参加企業からの紹介設計 | リファラルは費用対効果が最も高い |
| **P0** | ローカル SEO（ロングテール） | 「沼津 ホームページ 表示速度 改善」「沼津 ホームページ リニューアル 遅い」等の地域名 × 技術課題で上位化。ビッグKW（「沼津 ホームページ制作」）は比較サイトが壁のため正面突破しない | 個人事業が現実的に 1 位を取れる唯一の領域 |
| **P1** | 比較サイト掲載 | Web 幹事 / PRONI アイミツへ掲載申請 → AI 検索の一次情報として拾われる土台づくり | AI 検索の引用元は比較サイト経由が大半 |
| **P1** | ポートフォリオ可視化 | recruit キャンペーン（10 業種 × 1 社無料構築）の成果を実績として陳列。「業種数」でなく「成果数値」で見せる | 競合の弱点が「実績の中身が空」 |
| **P2** | note 毎日発信 | 業種別 SEO 記事・補助金教育コンテンツ・技術記事で AI 引用獲得 | 全国大手が書くがローカル沼津特化は空白 |
| **P3** | 広告 | 紹介が回り出すまでの繋ぎ。常用しない | 個人事業の定石 |

## 4. メッセージング規律

| 場面 | OK | NG |
|---|---|---|
| 技術品質の語り方 | 「表示速度を上げ離脱を減らし問い合わせを増やす。その根拠が機械検証スコア」 | 「Lighthouse 100 点」を結論として置く |
| 価格の見せ方 | 「機械検証付き・AI 最適化実装込みで 35 万円〜＝同価格帯テンプレ品と中身が違う」+ 補助金対応明示 | 価格表だけ提示 / 価格非開示 |
| 実績の見せ方 | 成果数値（問い合わせ件数・表示速度改善幅） | 「対応業種 ◯◯ 種」だけ |
| トーン | 事実 + 控えめな自負 + 数値根拠（HARTON 本サイトトーン） | 権威語・断定エピグラム・「独立認定機関」級の大袈裟さ |

## 5. 競合攻略ロードマップ

### 実競合（調査で特定 / 実在サイト）

| 競合 | 訴求 | 価格 | 構造的弱点 |
|---|---|---|---|
| リセラ（riselor.jp） | 伴走サポート | 38.5〜132 万 明示 | 技術スタック非公開・速度訴求なし |
| オフィスグルー（gluee.jp） | 地域密着・売上アップ | 数字なし | 実績ページの中身が空・成果根拠なし |
| ソフトメディア（soft-media.jp/numazu） | 顔の見える会社 | 非開示 | 自社サイト画像が placeholder のまま＝技術を実演できていない |
| トムズワン（toms1.net） | 成果のしくみ | 非開示 | 自社サイトがシンプル HTML でモダンさに欠ける |
| ロコプリオ等 格安勢 | 月 4,400 円〜 | 価格訴求のみ | 低価格ゆえ品質・専任性に疑問 |

**共通弱点 = 価格非開示・成果数値なし・自社サイトの技術品質を証拠化していない。**
T.C.HARTON は CWV 全 GOOD / WCAG AA / セキュリティ A+ / JSON-LD / llms.txt という「実演可能な証拠」を全部持つ — これが唯一の構造的優位。

### マイルストーン

| 期間 | アクション | 奪うポジション |
|---|---|---|
| **90 日** | Web 幹事・PRONI アイミツへ掲載申請 / 「沼津 × CWV・表示速度」記事 3 本 | 格安勢（ロコプリオ等）の品質不信層 |
| **6 ヶ月** | 業種別 LP 3 本投入 + Stella 実測ランキング公開（集計のみ） | トムズワン・オフィスグルーの「成果根拠なし」ポジションを技術証拠で侵食 |
| **1 年** | 「静岡県東部 WEB 制作」でトムズワン超え / 補助金教育コンテンツ展開 | リセラの高単価層に「同等品質 + 透明価格」で対抗 |

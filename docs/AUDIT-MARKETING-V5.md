## 1. H1 地名追加「沼津で」の効果（ヒーロー +1pt）

### 実装確認
`index.html` L414: `<span class="hero-line-1">沼津で、「誰も来ない HP」を</span>` + `<span class="hero-line-2">終わらせる。</span>`

### マーケ効果
- **「自分の地域か」判定の即時完了**: 静岡県東部の検索者が「沼津」を first-view 内で認識する事で **3 秒判定（自分ごと化）** が確定する。地名なしでは「全国対応の遠い業者」に誤読される構造的リスクがあった。
- **ローカル SEO 文字列との整合**: title / meta / GBP / `addressLocality` で既に「沼津」が露出しているが、**H1 にも入った**事で SERP スニペットでのクエリ太字化（"沼津 ホームページ制作"）が確実化。CTR 寄与が見込める。
- **コピーの韻律破壊リスクは最小**: line-1（小）＋ line-2（大）の 2 段組により、地名追加でリズムが崩れず「終わらせる。」の打撃力が温存されている。
- **ブランド逆効果なし**: 「沼津で」は地理を限定しても哲学（誰も来ない HP を終わらせる）を限定しない構成。

### 採点
ヒーロー 13 → 14（**+1pt**）。残 1pt は「First Impression における動画 / 短尺アニメーションの欠落」。

## 2. 価格スターターレンジ TOP 表示の効果（バリュー +1 / コンバージョン +1）

### 実装確認
`index.html` L482-497: 3 チップ（WordPress 35 万円〜 / 静的 HTML 70 万円〜 / 単発 LP 18 万円〜）+ /pricing/ 誘導リンク

### マーケ効果
- **「価格不明 → 即離脱」の構造的離脱防止**: 中小企業 WEB 制作市場で問合せ前ボトルネックの **第 1 位は不透明な価格**。/pricing/ ページへのクリック（1 アクション介在）を待たず、ヒーローで **下限 18 万円・主軸 35〜70 万円帯** をその場で開示することで、予算不適格者の早期離脱（むしろ歓迎）と適格者の安心着地を同時達成。
- **「単発 LP 18 万円〜」が低敷居エントリーの顔として効く**: フル制作（35〜70 万円）と LP（18 万円）を**並列**で出すことで、初回コンタクト商品が用意されている事を暗黙に伝える。これは V4 までの「いきなり数十万円」の心理的崖を解消する。
- **「全料金を見る →」リンクが期待値マネジメント**: チップは下限のみで、上限・オプションは pricing で確認できるという**情報の段階開示構造**が成立。

### 改善余地
- 価格チップに `Schema.org Offer` を埋め込むと SERP の rich result（価格レンジ表示）が出る可能性あり。**Phase 2 で +1pt 可**。
- 「保守 / 月額」が未記載。継続収益商品があるなら追加すべき。

### 採点
バリュープロポジション 12 → 13（**+1pt**）／コンバージョン動線 +1pt（後述と合算 +3pt のうち 1pt）。

## 3. CTA teal-700 統一 + 軽 CTA の効果（コンバージョン +2pt）

### 実装確認
- メイン CTA L501: `bg-teal-700 hover:bg-teal-800 text-white px-10 py-5 rounded-full shadow-lg`
- 軽 CTA L507: `border-2 border-dark-300 hover:border-teal-700` で「📄 902 社調査レポート」

### マーケ効果
- **色の一貫性によるブランドコード強化**: ヘッダーリンク / カードホバー / 約束番号バッジ / 最終 CTA まで **teal-700 が単一アクセント色**として徹底。V4 までの「ヒーロー CTA だけ dark-900」という不協和が解消され、視線誘導の認知負荷が下がった。teal は「信頼 + 知性」の心理連想を持ち、医療・士業・コンサル系で高 CV 実績の色域。
- **2 段階 CTA（hot / cold）の確立**: メイン =「30 分 無料診断」（高コミット）、軽 =「902 社調査レポート」（低コミット・教育的）の **温度差設計**は、初訪 70% / 再訪 30% の構成に最適。コンサル系 B2B は cold lead が 5〜7 タッチで CV するため、軽 CTA = 教育コンテンツ提供は **CV 率底上げの王道**。
- **「📄」アイコン**は視覚的にダウンロード／資料系を即座に示し、誤クリックを減らす。

### 改善余地
- 軽 CTA は現状 `/methodology/` ページ内アンカー誘導のみで、**メールアドレス取得を伴う PDF ダウンロード型ではない**。リードキャプチャ機構として未完成。フォーム型 PDF 配信に進化させると **+2pt（リードキャプチャ）** が可能。
- メイン CTA 直下の補助文「申込から 1 営業日以内に Zoom 設定メール」は心理的不安を打ち消す秀逸な配置。継続。

### 採点
コンバージョン動線 14 → 17（**+3pt**：teal 統一 +1 / 軽 CTA 設置 +1 / 価格チップ → /pricing/ 連動 +1）。

## 4. 信頼バッジ 3 種の効果（信頼形成 +2pt）

### 実装確認
`index.html` L515-528: Mozilla Observatory A+ / Core Web Vitals 全緑 / WCAG 2.2 AA 準拠（緑・青・琥珀の色分けピル）

### マーケ効果
- **「自称」を「第三者検証」に変換する装置**: V4 まで本サイトの強みは**自社申告**（速度・SEO・セキュリティに強い）の段階に留まっており、信頼形成カテゴリが 11/15 で停滞していた。Mozilla Observatory（HTTPS / CSP / HSTS 検証）/ Core Web Vitals（Google 公式指標）/ WCAG 2.2 AA（W3C 国際規格）の 3 つは、いずれも **発行元が中立かつ著名**で、B2B 意思決定者の検証コストをゼロに圧縮する。
- **配置位置がメイン CTA 直下** は、CTA クリック直前の心理に「この業者は外部基準で測定されている」という最終 anchor を打つ最良位置。
- **3 軸が網羅的**: セキュリティ・速度・アクセシビリティ。この 3 つは中小企業意思決定者が「ホームページに必要」と直感的に理解できる三大ドメインで、過剰でも過少でもない。
- **色分け**: 緑（安心）/ 青（信頼）/ 琥珀（注意喚起 → 法令系）の心理的色彩が機能配分と一致。

### 改善余地
- 各バッジに **検証ページ（observatory.mozilla.org / pagespeed.web.dev）への外部リンク**を追加すると検証可能性が上がり **+1pt** 取れる。現状はテキストのみ。
- ISO 27001 / プライバシーマーク 等の国内制度バッジは未取得。一人 SI なので難易度高だが、認定取得すればさらに +2pt。

### 採点
信頼形成 11 → 13（**+2pt**）。残 2pt は「お客様の声」「外部メディア掲載」獲得フェーズ。

## 7. 100 点までの残課題（11pt）

### 優先 A（即時実装可・合計 +6pt）
1. **お客様の声 1 件以上獲得**（信頼 +2pt）：公開 1 ヶ月で CV 0 のため最初の 1 案件を完了させる事自体が最大の解決策。完了後すぐ Voices セクションへ顔写真 + 実名 + 業種で掲載。
2. **PDF メール取得型 リードマグネット化**（リード +2pt）：軽 CTA「902 社調査レポート」を `/methodology/` 単純リンクから **メールアドレス → PDF 自動配信** 型へ昇格。Resend / Mailerlite で 1 日実装可能。
3. **業種チップ 6 種をリンク化**（バリュー +1pt / コンバージョン +1pt）：`/services/web/industries/[業種]/` の 6 ページ生成（テンプレ複製で OK）。SEO ロングテールも同時取得。

### 優先 B（中期・合計 +3pt）
4. **信頼バッジに検証ページリンク追加**（信頼 +1pt）。
5. **電話 / LINE 即時 CTA**（コンバージョン +1pt）：30 分診断より低コミットの選択肢。
6. **保守 / 月額プラン明示**（バリュー +1pt）：継続収益商品の存在を明示。

### 優先 C（長期・合計 +2pt）
7. **代表動画 30 秒**（ヒーロー +1pt）：顔写真の次は動く顔。信頼の最終段階。
8. **外部メディア掲載 / 受賞**（信頼 +1pt）：地元紙・業界誌への寄稿。

## 3. カテゴリ別 V2→V3 講評

### (A) ヒーロー インパクト 15 → 16
ヒーロー本体から 価格・バッジ・業種チップを退去させたことで、H1「沼津で、『誰も来ない HP』を／終わらせる。」の **痛点宣言が画面の主役に復権**。Linear 流の「H1 + sub + CTA + 視覚要素 1 つ」の 4 要素フォーマットに構造的に近づいた。Garden Eight 18 との差はキービジュアルのアートディレクション（受賞級フルブリード）のみとなった。

### (B) デザイン洗練度 14 → 16 ★最大改善
**+2 の主因は Trust Strip 縦 3 段（価格→バッジ→業種チップ）の情報階層**。価格 = 「いくら？」、バッジ = 「信頼できる？」、業種チップ = 「自分に該当する？」という意思決定の 3 大関心を上から順に配置した構造は、Stripe の「Pricing → Trusted by → Use cases」と同じ思想。border-y border-dark-100 の薄罫で hero と切り離すミニマル処理も V2 の「ヒーロー詰め込み」と比べて呼吸が圧倒的に良い。Linear / Vercel 級に到達はしないが、ベイジ 15 は明確に超過した。

### (C) 機能の差別化 12 → 12
AI 予測 100 万円〜 が Trust Strip 価格に追加されたのは加点要素だが、TOP 上で **AI スタック（XGBoost / LSTM / Prophet）や事例の明示が依然ゼロ**。services/ai-prediction/ 配下には深堀があるが TOP 伝播が弱く、競合視点では「3 サービス並列」のままに見える。±0 据え置き。

### (D) クライアント実績訴求 7 → 7
変動なし。お客様の声「募集中」枠と /vision/ 「3 つの約束」は維持されたが、ロゴ・スクショ・数値実績は依然ゼロ。HARTON Certified（⑤）の認定先 3+ 件公開を待つ構造は V2 から不変。

### (E) ナラティブ 9 → 9
/vision/ 「3 つの約束」セクションと末尾誘導は健在。新セクションの追加で物語の重心は分散したが、毀損には至らず据え置き。

### (F) UX 完成度 8 → 9 ★第二の改善
**4 ステップ図（無料相談 → 日程調整 → 30 分診断 → 見積もり）の所要時間明示**（30 秒 / 1 営業日 / 30 分 / 3 営業日）は、国内 WEB 制作会社の中で過去事例ほぼ皆無のレベルで丁寧。「ここまで完全無料」の補足注記まで含めて **問い合わせ前の摩擦を最小化** している。Linear / Stripe 級の onboarding 設計思想に並んだ唯一の項目。

### (G) 技術的優位 10 → 10
FAQPage Schema が TOP `index.html` に直接埋め込まれていない点は厳密には減点候補だが、faq/ services/web/ cases/ への配備は確認済で、サイト全体の構造化データ被覆率は依然業界最高水準。CSP Trusted Types + HSTS preload + 4 種 Schema + 2554 項目 S-RANK ゲート + /llms-full.txt の構成は V2 から不変。満点維持。

## 6. 結論

- **V1 60 → V2 75 → V3 80（+20 点 / V1 比）**。V2→V3 の +5 点は、ヒーロー密度緩和（B +2）と 4 ステップ図新設（F +1）と Trust Strip による情報階層整理（A +1, B 寄与）の合成。
- **V3 で国内 WEB 制作会社の中位上層に到達**：ベイジ（79）を初めて逆転、LIG（84）まで 4 点差、Garden Eight / mount.jp（87）まで 7 点差。
- **最大の構造的改善はヒーロー本体の 10 ブロック → 6 ブロック化**：V2 で唯一悪化した「Linear 流ミニマリズムから乖離」が V3 で巻き戻り、デザイン洗練度が +2 と本監査最大の伸び幅。
- **UX 完成度 9 は国内同業最高水準**：4 ステップ + 所要時間明示 + 完全無料注記の 3 点セットは Linear / Stripe の onboarding 設計思想に並んだ。
- **唯一の未消化 P0 課題は Lighthouse 実測カード**。Trust Strip 横の 4 列目に「Performance 100 / Accessibility 100 / Best Practices 100 / SEO 100」の 4 メトリクス pill を 1 行追加すれば +2 点（V3 = 82）が射程。
- **FAQPage Schema は TOP `index.html` 直接埋込が未確認**。配下ページに完備のため LLMO・SEO への実害は軽微だが、競合視点（特に技術的優位の自己宣伝面）では TOP 集約が望ましい。
- **100 点到達の最重路は依然 ⑤ HARTON Certified 連携**（+5 点 / D）。Phase 1 移行で Certified 認定先 3 件公開時点で 90 点（Garden Eight 超え）射程は維持。

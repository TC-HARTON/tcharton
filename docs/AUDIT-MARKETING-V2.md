## 1. ヒーロー & First Impression（10/15・+2）

### 改善された点
- **プロモバナー撤廃により信頼感が回復**。「残り 2 枠」という人為的希少性は B2B 中小企業層にとって「胡散臭さ」直結ワード。撤廃判断は正しい（+2pt）
- ジャーゴン削除で読者語彙との距離が縮まった（HARTON Stella / Deep Work などの内部用語が消えた）

### 残課題
- H1「誰も来ない HP を終わらせる」は依然として **読者語彙ではない**。沼津の社長は「ホームページから問合せが来ない」「売上が上がらない」と言う
- ヒーロー右の `hero-evidence`（902 件・3.8 倍・大内直接対応）は**良い証跡**だが、フォントが小さく（text-xs）視線到達率が低い
- Old/New/AI の SVG アニメは情緒的には◎だが「+47 件 / +28%」が **架空の数値**として表示されており、お客様の声不在と相まって「絵空事」リスクを残す

### 推奨改善（→ 13/15 まで上昇可能）
1. H1 を「**沼津で『HP から問合せが来ない』を終わりにする**」に書換（地名 + 読者語彙）
2. `hero-evidence` を text-sm に格上げ + アイコン化
3. SVG の「+47 件 / +28%」を**実数値が出るまで非表示**、または「業界中央値との差」など出典のある数字に置換

## 2. バリュープロポジション（9/15・+2）

### 改善された点
- 3 サービスカード（WEB 制作 / AI 予測 / 私たちの想い）の説明文がジャーゴンから読者語彙へ近づいた
- 「3 つの約束」セクション追加により**「何を約束するのか」が明文化**された（V1 では『S クラス』という未定義語のみだった）

### 残課題
- 「3 つの約束」の文章が**抽象**：「数値で品質を約束する」と書きながら、その**数値が直後で示されない**。約束を訴求した直後に「LCP < 2.5s / アクセシビリティ AA / OWASP Top10 ゼロ違反」など**具体数値の固有名詞**を 1 行で出すべき
- 3 サービスの**価格レンジが TOP に無い**（30〜80 万 / 月 1〜3 万 / 100〜300 万 が JSON-LD のみ）。価格は B2B 検討の最初のフィルタ
- 「私たちの想い」カードはバリュープロポジションでなく**情緒**。3 つ並べる場合は「Why Us」型の差別化軸（沼津唯一・代表直接・保守込み）を立てる方が CV 直結

### 推奨改善（→ 12/15）
- 「3 つの約束」内に各 1 行の**具体数値**を追加（毎月品質再判定 → "Lighthouse 90+ / Core Web Vitals 緑帯"等）
- TOP に**価格スターターレンジ**を出す（"WEB 制作 30 万円〜・保守 月 1 万円〜"）

## 3. コンバージョン動線（14/20・+6・★最大改善）

### 改善された点
- **最終 CTA セクション追加**は単独で +4pt の威力。スクロール下端まで読んだ訪問者の離脱を救う最重要施策
- TOP に CTA が**ヒーロー / リードマグネット / 最終 CTA の 3 段配置**となり、滞在中のどの位置でも CTA に手が届く構造に進化
- プロモバナー撤廃により**「焦らされてる感」が消失**し、純粋な意思決定動線になった

### 残課題
- 全 CTA が「**30 分無料診断**」一本足打法。心理的ハードルが高い訪問者向けの**ライト CTA**（資料 DL / メルマガ / 質問だけ送る）が依然として無い
- CTA ボタンの色が `bg-dark-900`（黒）。teal-700 のブランド色を CTA に使う方が視認性・記憶性の点で勝る（A/B テスト推奨）
- ヒーローの 3 サービスカードが**サービス紹介ページへの遷移**であり、**CV ページへの遷移ではない**。Hick の法則上、選択肢が分散している

### 推奨改善（→ 17/20）
1. 「ライト CTA」追加：**資料 DL（メアドのみ） / 60 秒で見積もり / LINE で質問**
2. CTA 色を teal-700 に統一（ブランド一貫性）
3. ヒーローカードに「**まず話を聞く**」CTA を 4 枚目として追加

## 4. コンテンツ深度（12/15・+1）

### 改善された点
- リードマグネットセクションで「業種別の傾向（飲食 / 建設 / 美容 / 製造）」が訴求対象として明示された
- ジャーゴン削除により methodology / about / profile への遷移がスムーズに

### 残課題
- 16 ページのコンテンツ密度は依然 80% 水準。業界別ランディング（飲食業向け / 建設業向け）が無い
- 改修で TOP は厚くなったが、**深度のあるページ（cases / 業種別事例）への動線**は弱い

### 推奨改善（→ 14/15）
- TOP に「業種別の傾向」を 4 タイル（飲食 / 建設 / 美容 / 製造）で配置 → リードマグネットへ集約

## 5. 信頼形成（9/15・+3）

### 改善された点
- 「お客様の声（準備中）」セクション追加は**勇気ある正直訴求**。隠すよりも「最初のお客様には特別なサポート」と能動メッセージ化したことで、不在を強みに転化（+2pt）
- footer h2→h3 で見出し階層が正常化、E-E-A-T シグナル改善（+1pt）

### 残課題（依然致命部）
- **顔写真が TOP に無い**（profile/ には存在）。代表直接対応の最大の証拠は「顔」
- **ロゴ / クライアント実績が無い**（公開 1 ヶ月のため不可避だが、過去の業務実績の社名 or 業種ロゴでも可）
- **第三者保証**（GitHub stars / Lighthouse スコア / Cloudflare Trust Score 等）が TOP に無い
- 「902 社調査」の**生データ・PDF サンプル画像**が無い

### 推奨改善（→ 13/15）
1. ヒーロー右上に**代表大内達也の顔写真（円形・60-80px）**配置
2. 「**過去の実績（業種ロゴ）**」セクション追加（個人特定不可形式でも可）
3. methodology の Lighthouse スコア / セキュリティスコアを TOP にバッジ表示

## 6. リードキャプチャ（7/10・+5・★致命部修復）

### 改善された点
- リードマグネットセクションは V1（2点）から劇的改善。**902 社レポート / PDF 12 ページ / 4 つの中身予告**は B2B での王道訴求
- 「PDF / 12 ページ / 無料」のスペック明示は信頼の核

### 残課題
- レポートの**実態が methodology ページの全文公開**であり、**メアド入力ゲートが無い** → リード（メアド）が取れていない
- ニュースレター登録 / RSS / LINE 公式 等の**継続接触チャネル**ゼロ
- 「※ 現在は methodology ページで全文公開中」という注記は誠実だが**マーケ的には機会損失**

### 推奨改善（→ 10/10）
1. **メアド入力フォーム**を設置し PDF を別資産として配信（最低限のリード化）
2. **note** 既出記事をニュースレター化、購読フォームを TOP に
3. メアド入力 → 翌日 1 通目「自社サイトの簡易診断ガイド」自動配信のドリップ設計

## 7. ブランディング（7/10・+2）

### 改善された点
- ジャーゴン（HARTON Stella / Deep Work / S クラス制作パートナー）の削除により**専門用語の壁**が低下。中小企業層の語彙に近づいた（+2pt）

### 残課題
- ブランドカラー（teal-700 / #1B4965 ディープティール）の使用が**散発的**。CTA は黒（dark-900）で統一されておらず、色のブランド訴求力が弱い
- `Sクラス保証` のスローガンが JSON-LD に残るが、TOP コピー上には実態なし → **構造化データと表示テキストの乖離**
- ロゴが SVG の T 一文字。記憶残存度が低い（mount.jp / garden-eight クラスとは差大）

### 推奨改善（→ 9/10）
- CTA 色を teal-700 に統一
- 「S クラス保証」の意味を TOP で **1 文定義**するか、表記そのものを削除
- ロゴをワードマーク化（T.C.HARTON のタイポグラフィ的処理）

## 9. 結論

### 改修評価：成功（B 判定）

| 観点 | 評価 |
|---|---|
| 当初の致命部修復 | ◎ 完全修復（リードキャプチャ +5、CV 動線 +6、信頼形成 +3） |
| 構造的 CV 阻害の解消 | ◎ プロモバナー撤廃 + 最終 CTA + リードマグネットで動線完成 |
| ジャーゴン除去 | ◎ 中小企業層との語彙距離縮小 |
| 平均サイト水準到達 | ○ 68 点は「平均的 B2B 業者サイト」 |
| 差別化 A 級到達 | ✕ 17pt 不足（顔・声・数字の三本柱が未完） |

### 47 → 68 の意味

**V1（47 点）の状態**：1000 人来ても CV 0 の構造（プロモバナー詐欺感 + リードマグネット皆無 + 最終 CTA 皆無）。流入を増やす意味が薄い。

**V2（68 点）の状態**：1000 人来れば 1〜3 件の CV が見込める動線が完成。流入施策（SEO / 広告 / note）を回す前提が整った状態。

### 次の Phase 目標

68 → 85（A 級）。最大の鍵は **「顔・声・数字」三本柱の実装**：
1. 顔 ＝ 代表大内達也の顔写真を TOP に
2. 声 ＝ 1 号顧客の獲得とインタビュー実装
3. 数字 ＝ 902 社レポートのメアドゲート化と実 CV 数のサイト掲示

これら 3 つを揃えれば B2B サイトとしての**信頼閾値**を超え、純広告流入でも CV する構造になる。

**監査者所感**：
> プロモバナー撤廃の意思決定は正しい。「残り 2 枠」は短期 CV を奪う代わりに**長期信頼を毀損**する典型的アンチパターンであり、撤廃により本サイトは「正直な専門家のサイト」というブランド軸を取り戻した。
> 一方、47 → 68 は「**致命的な穴の修復**」であり、依然として「**平均**」。次は『**他社にない何かを、TOP の上半分で 5 秒で伝える**』フェーズ。それは『代表大内達也の顔と、902 社調査の生データと、1 号顧客の声』の三点セット以外にあり得ない。

## 0. 議論の決着（3 派の統合）

3 派の対立は「どれが正しいか」ではなく「どの順序か」の対立だった。決着:

```
【論理的依存関係】
ポジショニング（何屋か）
   ↓ 定義する
顧客心理／メッセージ（誰の・どの不安に・何と言うか）
   ↓ 中身を決める
集客チャネル（どこで言うか）
```

```
【実行の緊急度】
集客チャネル派の指摘が運用上は最優先の事実：
「T.C.HARTON は比較ポータル非掲載 ＝ 沼津・三島の発注者の検討リストに存在しない」
→ ポジショニングが完璧でも、検索接点ゼロなら誰にも見られない無人地帯
```

**確定方針：ポジショニングと顧客心理は『先に設計を固める』。チャネルは『設計を待たず即着手』。**
3 つは対立しない。設計（ブランド・心理）と着手（チャネル）を並走させる。

## 1. ポジショニング（確定）

| 項目 | 確定内容 |
|---|---|
| ポジショニングステートメント | **沼津・三島で、"作って終わり"にしない唯一の WEB エンジニアリング事務所。** |
| カテゴリ | 「制作会社」ではなく「WEB を資産として設計・計測・改善する技術者」。競合7社は全員「制作屋」= 同じ土俵に乗らない |
| 上位概念 | **「データサイエンスで WEB を科学する事務所」**。WEB制作=実装 / Stella=計測 / AI予測=応用 を「科学する」の配下に階層化（3 事業の散漫さを解消） |
| 主訴求 | 「AI に、選ばれる WEB を。」（技術品質の顧客語への翻訳 = 約束が主役、技術は裏付け） |
| 信憑性の担保 | 代表のデータサイエンス15年が「科学する」統合の根拠 |

## 2. 顧客心理・メッセージ規律（確定）

### 沼津・三島の中小オーナーの本音（購買は「期待」でなく「恐怖回避」で動く）

1. この金額を払って本当に売上が上がるのか分からない
2. 専門用語で煙に巻かれて言い値で買わされるのが怖い
3. 作って納品したら連絡が取れなくなるのでは
4. 個人事業＝1人。病気・廃業したらサイトごと宙に浮く

### 先に潰すべき不安（AI・技術品質を語る「前」に伝える）

| 不安 | 応答（伝えるべきこと） |
|---|---|
| 逃げそう | 契約書・納品後サポート範囲の明文化・継続契約の保証条項 |
| 言い値不安 | 見積もりの内訳を素人語で全公開 |
| 成果が不明 | 「何をもって成功とするか」を発注前に合意する診断プロセス |
| 個人事業の不安 | 「代表が最初から最後まで直接担当。営業と制作者が別人で伝言ゲームにならない＝責任の所在が 100% 明確」を強みとして堂々と売る |

### メッセージの順序規律

```
①「逃げない・透明・責任明確」（信頼の器をつくる）
   ↓
②「AI に選ばれる WEB」（約束＝主訴求）
   ↓
③ 機械検証スコア・CWV・AI予測（裏付け＝二次）
```

逆順で語らない。技術から入ると沼津のオーナーには「営業トーク」に聞こえる。

## 3. 顧客ペルソナ（確定 / 4 類型）

3 派が提案した 7 ペルソナを重複統合し 4 類型に確定。

### ペルソナ 1：事業承継・刷新型（最重要）
- 沼津・製造/卸の二代目経営者、37〜45 歳、WEB リテラシー中〜高
- 動機：事業承継で会社の顔を一新したい / 先代サイトが 10 年放置
- 不安：安い業者はテンプレ、高い業者は何が違うか不明 / また放置されないか
- 決め手：見積もり内訳の透明性 + 代表本人と直接話せる距離感 + "作って終わりにしない"約束
- 刺さる訴求：「次の 10 年持つ作りか」

### ペルソナ 2：再制作検討型
- 三島・クリニック/整骨院、42〜52 歳、WEB リテラシー中
- 動機：3 年前に数万円で作ったサイトが崩れている / 競合が MEO で上に出た焦り
- 不安：高い金を出して結果が出ないこと
- 決め手：「何が達成できたら成功か」を発注前に数字で握る診断プロセス
- 流入：ローカル SEO「ホームページ 作り直し 三島」→ /stella/ 無料診断

### ペルソナ 3：採用難・AI 危機感型
- 沼津・三島の地場サービス業オーナー、45〜55 歳
- 動機：「AI や検索に見つけてもらえないと若い人材も客も来ない」という直感的危機感
- 不安：安い 3 ページサイトでは危機感が解消されない
- 決め手：「AI に選ばれる」が刺さる唯一の層。先行投資の価値を理解する
- 刺さる訴求：主訴求「AI に、選ばれる WEB を。」が直撃

### ペルソナ 4：放置サイト屈辱型（リテラシー低）
- 沼津・水産加工卸など老舗、55〜65 歳、WEB リテラシー低
- 動機：取引先の若手に「ホームページ古いですね」と言われた屈辱
- 不安：また放置される / 専門用語が分からない
- 決め手：「更新も私が責任を持って続けます」の一言 + 月額の中身が明確なこと
- 注意：技術・AI を語らない。「逃げない・分かりやすい」だけで決まる

## 4. 集客チャネル運用（確定 / 優先順位）

| 優先 | チャネル | 着手タイミング | 確定理由 |
|---|---|---|---|
| **P0 即着手** | 比較ポータル掲載（Web幹事 / PRONIアイミツ / 優良WEB） | 設計を待たず今すぐ | 検索上位をポータルが占有。非掲載＝検討リストに存在しない。ポータルは「入口」であり「決着の場」ではない。一覧で技術品質を掲げる唯一の事業者になれる（競合7社全員が空白） |
| **P0 即着手** | ローカル SEO（地域名 × 技術課題ロングテール） | 今すぐ | 「沼津 ホームページ 表示速度 改善」「ホームページ 作り直し 三島」等。ビッグKWは比較サイトが壁のため正面突破しない。個人事業が現実的に1位を取れる唯一の領域 |
| **P1** | ポートフォリオ可視化 | 設計確定後 | recruit キャンペーン成果を「業種数」でなく「成果数値」で陳列。競合の弱点が「実績の中身が空」 |
| **P1** | 紹介導線設計 | 設計確定後 | 費用対効果は最高だが効くのは数ヶ月後。設計（信頼の器）が固まってから |
| **P2** | note 毎日発信 | 並行継続 | 業種別 SEO 記事・補助金教育コンテンツ。ローカル特化は全国大手が書かない空白 |
| **P3** | 広告 | 紹介が回るまでの繋ぎ | 常用しない |

## 5. 競合攻略ロードマップ（確定）

### 沼津・三島 実競合 7 社（実地調査済）と共通弱点

| 競合 | 拠点 | 価格 | 強み | 弱点 |
|---|---|---|---|---|
| リセラ | 沼津 | 制作33万〜明示+運営サブスク | エリア最有力・丸投げ安心感・ポータル両掲載 | 技術スコア訴求ゼロ・サブスク前提が重い |
| オフィスグルー | 沼津 | 6.6万〜+ページ積上げ | 看板まで地場フルサービス・地域認知 | 画像最適化不足・実績不透明・技術で完敗 |
| ソフトメディア | 清水町 | 非開示 | ドローン/360°撮影が独自・沼津三島両LP | 価格非開示・差別化が「ワンストップ」横並び |
| トムズワン | 伊豆 | 非開示 | ニッチ業種の蓄積 | 沼津密着度中・デザインが古典的 |
| PC11 | 三島 | 非開示 | コンテンツマーケ志向・発信量多 | 実績2件・**HARTONに最も近い競合** |
| NEXT FUTURE | 清水町 | 3ページ5万円〜 | 価格の安さ+実績150社+対面営業力 | 薄利量産型・作り込み期待薄 |
| システムエイド | 沼津 | 基本5.5万+ページ1.98万〜 | 創業25年の信頼・Web幹事筆頭推し | 技術トレンド対応が弱いと推測 |

**競合 7 社共通の構造的弱点（＝T.C.HARTON の唯一の構造的優位）：**
全社が CWV / WCAG / セキュリティ A+ / JSON-LD / llms.txt / AIO を**一切訴求していない**。実績は「件数・業種・スクショ」止まりで成果数値が希薄。
T.C.HARTON はこれら「実演可能な証拠」を全部持つ。

| 期間 | アクション | 奪うポジション |
|---|---|---|
| 90 日 | Web幹事・PRONIアイミツ掲載申請 / 「沼津 × 表示速度・CWV」記事3本 / 見積もり内訳の素人語全公開ページ | NEXT FUTURE・システムエイド等 格安勢の品質不信層 |
| 6 ヶ月 | 業種別 LP 3本（製造・クリニック・工務店） / Stella 実測ランキング公開（集計のみ） / 発注前診断プロセスのページ化 | トムズワン・オフィスグルーの「成果根拠なし」層を技術証拠で侵食 |
| 1 年 | 「静岡県東部 WEB 制作」上位化 / 補助金教育コンテンツ / PC11 の技術志向層を「実演できる技術」で奪取 | リセラの高単価層に「同等品質 + 透明価格」で対抗 |

## 6. 価格戦略（確定）

- WordPress 35 万円〜 は静岡県中央値帯（57万）で妥当。だが沼津の「数万円〜」表示と並ぶと「高い」と誤認されるリスク
- **対策：価格そのものでなく「検証可能な品質指標」で土俵をずらす。**「機械検証付き・AI 最適化実装込みで 35 万円〜＝同価格帯テンプレ品と中身が違う」
- 見積もり内訳を素人語で全公開（顧客心理派の「言い値不安」解消と直結）
- 小規模事業者持続化補助金（Web 費上限 50 万）対応を明示

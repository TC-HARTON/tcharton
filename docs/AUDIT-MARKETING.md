## 1. ヒーロー & First Impression（8/15）

### 1.1 現状（index.html L412-491）

```
H1: 「誰も来ない HP」を、終わらせる。
sub: 高品質 WEB サイトを、静岡から発信。作って終わらない WEB 制作と、AI 予測。
証跡: 静岡県 902 件調査 · 業界平均 3.8 倍品質 · 代表 大内 達也 直接対応
CTA: 30 分 無料診断を受ける
```

### 1.2 評価

**良い点 (+)**:
- H1 が「痛み起点」になっている（mount.jp / garden-eight より痛みが鋭い）
- 静岡 902 件・3.8 倍 という独自数値証跡が存在する
- アニメーションで Old → New の物語を見せようとしている（着想は◎）

**致命的問題 (-)**:

| # | 問題 | 影響 |
|---|---|---|
| H-1 | **H1 が抽象的** —「誰も来ない HP」は読者の課題語彙ではない（読者は「ホームページから売上が上がらない」「問合せがない」と言う） | -3 |
| H-2 | サブコピーが**機能列挙**（WEB 制作 / AI 予測）で、**結果のベネフィット**が無い | -2 |
| H-3 | ヒーロー直下の「3 リンクカード」が WEB / AI / 想い と**並列**で、**訪問者の意思決定を分散させる**（Hick の法則違反） | -1 |
| H-4 | アニメーションが情報過多で 5 秒の理解を遅らせる（古ブラウザ → 粒子 → 新ブラウザ + +47 / +28% など） | -1 |
| H-5 | H1 と CTA「**30 分無料診断**」が**論理接続していない**（誰も来ない HP を終わらせる、と、30 分診断、の間に距離がある） | -1 |

### 1.3 競合 5 サイト並列比較

| サイト | H1（推定/参考） | 強み |
|---|---|---|
| **mount.jp** | "Designing the Future of Brands." | 抽象的だが**視覚的世界観で押し切る** |
| **garden-eight.com** | "We Build Future Heritage." | 詩的・記憶性 |
| **Stripe** | "Financial infrastructure for the internet" | **対象 + 機能 + 規模感** が 1 行 |
| **Linear** | "Linear is a purpose-built tool for planning and building products" | **誰のための何か**が即座にわかる |
| **Hey** | "Email at its best, finally" | **痛み + 解放感** |

**tcharton.com の現 H1 は、Hey 型（痛み起点）を狙っているが「誰も来ない HP」が読者語彙でない**ため共感が弱い。

### 1.4 改善案

**改善 H1 候補（A/B 案）**:

> **A 案（痛み + 結果）**:
> 沼津で「ホームページから問合せが来ない」を、終わりにします。
> ~~静岡 902 社調査~~ で見えた、CV を生む WEB の作り方。

> **B 案（数字 + 約束）**:
> 静岡 902 社の WEB を解剖した男が、
> あなたの 1 社だけ、Sクラスで作ります。

> **C 案（恐怖 + 緊急）**:
> あなたの HP は、Google で見られる前に
> ChatGPT で素通りされています。
> 沼津から AI 検索時代の WEB を。

**サブコピー改善**:
> ~~高品質 WEB サイトを、静岡から発信。作って終わらない WEB 制作と、AI 予測。~~
> → **「制作して終わり」を許さない月額保守 + 売上を予測する AI モデル。沼津・三島・富士の中小企業オーナー専属、代表 大内が一人で全工程を担当します。**

**ファーストビュー再設計**:
- 左 60% : 大内氏の顔写真 (現在 contact ページ最下部に小さく埋もれている **CEO 写真を ヒーローに昇格**)
- 右 40% : H1 + サブ + CTA + 緊急バナー
- アニメは hero 下のスクロール後に移動（一瞬の理解を阻害しない）

## 2. バリュープロポジション（7/15）

### 2.1 現状訴求の解析

サイト全体から抽出すると、訴求は **5 軸混在**:
1. 静岡 902 社調査（独自データ）
2. Sクラス保証 / 機械検証 2554 項目
3. 代表 大内が一人で対応（属人）
4. WEB + 保守 + AI 予測 の 3 サービス（範囲）
5. 月額に縛られない買い切り（価格哲学）

### 2.2 問題

- **5 軸あって**、訪問者は「で、結局この会社は何屋？」となる
- 「Sクラス保証」「2554 項目」「★★★」など**自社造語**が多く、**意味伝達コスト**が高い
- 「静岡 902 社調査・3.8 倍品質」は強いが、**CV する人にとっての利得語**に変換されていない（「3.8 倍品質 = 何が自分に良いのか？」）

### 2.3 改善案 — 単一の Position Statement に集約

```
For: 沼津・三島・富士の中小企業オーナー（年商 5,000 万 〜 5 億）
Who: 「自分のビジネスを誰よりわかってくれる WEB 担当者」が欲しい
Our: T.C.HARTON は、
     一人の専門家による「Sクラス保証」WEB 制作 + AI 予測サービス
Unlike: 月額固定でフロー納品しかしない地元制作会社・東京の安価テンプレート業者
Our service: 静岡 902 社調査の上位 1% 品質を、買い切りで届け、AI で売上を予測する。
```

**1 ページ目に置くべき Hero 下セクション** ↓

> **なぜ沼津の中小企業オーナーが、私たちを選ぶのか。**
>
> 1. **静岡県 902 社の WEB を分解した実測データ**で、競合より何が劣っているか具体に診断します。
> 2. **代表 大内が一人で全工程**を担当。下請け・営業・ディレクター不在で、伝言ゲームゼロ。
> 3. **買い切り + 任意月額保守**。月額に縛られず、必要な時だけ伴走します。
> 4. **AI 売上予測モデル**で、HP 公開後の "なんとなく更新" を "数字で更新" に変えます。


### 5.3 改善案 — 90 日緊急アクション

**Week 1-2 — 種を撒く**
1. 知人経営者 5 名に**完全無料**で WEB 構築 → 顔写真 + 推薦動画（30 秒）取得
2. 自社診断レポート（PDF）を 30 社に**無料送付** → 「使ってよかった」コメント取得
3. note の文章を 8 本/月 → **被引用 / シェア**を獲得しサイトに埋込み

**Week 3-4 — 構造化**
4. ヒーロー下に「制作実績ロゴ 6 社」（最初は無料案件 + 知人会社で OK）
5. profile に**経歴の数字化**: 「○○年制作実績 △件」「○年機械学習開発 △件」
6. お客様の声 3-5 件 を**実名 + 顔写真 + 動画**でランディング各所に埋込み

**Month 2-3 — 第三者評価**
7. **沼津商工会議所・静岡新聞ビジネス欄**に取材依頼（プレスリリース）
8. **G's ACADEMY / Reskilling 系メディア** に寄稿で被リンク + 権威獲得

### 6.2 改善案 — リードラダー設計

**Tier 1（無料・摩擦最小）**: メール 1 項目で取得
- 📄 **「静岡県 902 社 WEB 品質調査レポート」PDF 無料 DL** ← **既に方法論にデータ存在 / 即実装可**

**Tier 2（中摩擦）**: 詳細情報入力で取得
- 📞 **「現サイトの SEO/GEO 機械診断レポート（手動 1 枚）無料」** ← URL 入力 + メール
- 📅 **「無料 15 分（30 ではなく 15）相談予約」** ← Calendly 直結

**Tier 3（高摩擦・既存）**: 30 分無料診断・正式見積

```html
<form>
```

## 8. 沼津の中小企業オーナーが見たときの離脱ポイント TOP 5

1. **「Sクラス」「★★★」「2554 項目」「機械検証」「GEO」「LLMO」など、知らない単語が多すぎる** → 1-2 行スクロールで離脱
2. **顔が見えない**（hero に大内氏が居ない / 連絡先電話番号も見えない）→ 「どこの誰だ」で離脱
3. **料金が「○○ 万円〜」でしか出ていない場合の上限不安** → 「うち、いくらになるの」で離脱
4. **電話番号が無い** → 60 代経営者は電話で問合せたい・存在しない時点で「実体不明会社」判定
5. **問合せフォームのフィールドが多い**（氏名・会社・メール・選択・詳細）→ 入力途中離脱

### 即対応案
- ヘッダー右側に **電話番号** を明示（受付時間付き）
- 料金ページに**実例 3 件の総額表**（A 社: 飲食 5P / 50万 など）
- フォームを**メールのみ→詳細は返信時**の段階方式に

① WEB 構築（15 問）

Q1. なぜ買い切り型なのですか？
月額制は数十万円分の支払いを 5 年で続ける構造です。当社は買い切りで 30〜80 万円を一括（または分割 2〜3 回）でお支払いいただき、その後の保守運用は月額 1〜3 万円のプランと切り分けます。クライアント様は早期に投資回収でき、サービス契約に縛られない自由を保てます。
Q2. 月額制と比べて本当にお得ですか？
5 年スパンで見ると、月額制（月 3 万 × 60 ヶ月 = 180 万円）に対し、買い切り 50 万 + 保守スタンダード 60 ヶ月 = 132 万円となり、累計でも安価です。所有権がお客様にある点も大きなメリットです。詳細は /pricing/ の LTV 試算をご覧ください。
Q3. 制作期間はどのくらいですか？
標準で 4 週間（W1: ヒアリング・設計、W2: デザイン・コーディング、W3: 実装・テスト、W4: 検証・公開・引継ぎ）です。業種特性や写真撮影が必要な場合は 5〜6 週間となることもあります。
Q4. 自分でも更新できますか？
Markdown ベースの編集インターフェースをご希望に応じて構築可能です。ただし WordPress のような汎用 CMS は採用しません（保守負担とセキュリティリスクが大きいため）。日常的な軽微更新は保守プランで対応する設計を推奨しています。
Q5. スマホ対応は標準ですか？
標準対応です。モバイルファーストデザイン + Core Web Vitals 最適化 + WCAG 2.2 Level AA 準拠で構築します。Tailwind CSS のレスポンシブクラスで全画面サイズを網羅します。
Q6. ドメイン・サーバーはどうなりますか？
ドメインはお客様名義で取得・管理いただきます（T.C.HARTON では代行管理しません）。サーバーは Cloudflare Pages（無料枠で十分なケースが多い）または希望のホスティングを選択可能です。
Q7. 既存のメールアドレスは使えますか？
ドメインメール（info@yourdomain.com 等）の継続利用は可能です。Cloudflare Email Routing や Google Workspace 等との連携を構築時にご提案します。
Q8. 業種特化のデザインは可能ですか？
GEO-STANDARDS §4.1 のドメイン分類（Law & Government / Opinion / Statement 等 10 ドメイン）に基づき、業種ごとに最適な GEO 戦略を選定して構築します。詳細は /services/web/industries/ をご覧ください。
Q9. 写真撮影は別料金ですか？
撮影同行は別料金（半日 3.3 万円・沼津/三島/富士エリア）です。基本構築には含まれません。お客様撮影の写真または無料素材を活用するケースが多いです。
Q10. 原稿作成は手伝ってもらえますか？
中規模プラン（50 万円帯）では原稿の構成サポート（叩き台作成・校正）を含みます。完全な代筆は別料金（11,000 円〜 / 1,000 字）で対応可能です。
Q11. 制作後の小さな修正はできますか？
保守プラン（スタンダード以上）に含まれる軽微修正で対応可能です。プラン外の場合はスポット作業（時間単価 5,500 円）で承ります。
Q12. SEO 対策は含まれますか？
構築時の標準対応として Google 検索セントラル準拠の SEO（title / description / OGP / sitemap.xml / robots.txt / 構造化データ等）を含みます。継続的な SEO 最適化は保守プランで対応します。
Q13. 多言語対応は可能ですか？
可能です。日本語 + 英語の 2 言語が標準で、追加言語も対応できます。hreflang 属性 + 言語別 sitemap 構築含めて大規模プラン（80 万円帯）で承ります。
Q14. EC サイトは作れますか？
Shopify / Stripe Payment Links / Cloudflare Stream 等を組み合わせた小〜中規模 EC は対応可能です。在庫数千 SKU 以上の本格 EC は別途要件定義となります。
Q15. 既存サイトのリニューアルも可能ですか？
可能です。リニューアル時は既存コンテンツの 301 リダイレクト設計、SEO 順位の維持、Search Console 移行手続まで含めて対応します。

② 保守運用（10 問）

Q16. 保守運用は必須ですか？
必須ではありません。ただしセキュリティ脆弱性は毎月新規発見されます（OWASP / IPA 参照）。放置サイトは半年で表示速度が劣化することも一般的です。月額 1 万円のライトプランから提供しています。
Q17. プランの違いは何ですか？
ライト（1 万・監視のみ）/ スタンダード（2 万・月 60 分修正可）/ プレミアム（3 万・月 3 時間修正 + SEO/LLMO フル対応）の 3 段階です。詳細は /services/maintenance/plans/ で完全比較表を公開しています。
Q18. 解約はいつでもできますか？
はい。解約申請から 1 ヶ月後に終了します。違約金なし・縛り月数なし。データはすべてお客様にお渡しします。
Q19. プラン変更の方法は？
メール 1 通でいつでもアップ・ダウン可能です。翌月から新プランが適用されます。変更回数の上限はありません。
Q20. 緊急時の対応はどうなりますか？
ライトは通常対応、スタンダードは 24 時間以内、プレミアムは即日対応です。プラン外の緊急対応はスポット 11,000 円〜 / 件で承ります。
Q21. 月次レポートは見るのが大変では？
サマリー（30 秒で読める）+ 提案アクション（お客様判断箇所のみ強調）の構成です。実物見本を /services/maintenance/report-sample/ で公開しています。
Q22. 軽微な修正の範囲は？
テキスト修正、画像差替え、お知らせ追加、リンク修正等の数十分以内で完了する作業です。スタンダードは月 60 分、プレミアムは月 3 時間が枠です。
Q23. 大きな変更は別料金になりますか？
プラン枠を超える作業はスポット作業（時間単価 5,500 円）または機能追加見積（別途）となります。事前見積り承認後に着手します。
Q24. 他社で作ったサイトの保守も可能ですか？
可能です。ただし既存サイトの構造によっては Sクラス保証の品質に到達できないケースがあります。Phase 1（無料診断）で現状を可視化し、保守可能性をご報告します。
Q25. ドメイン・サーバー管理も含まれますか？
保守プラン全プランで Cloudflare Pages の管理・SSL 証明書更新監視を含みます。ドメイン更新は名義人（お客様）の自動更新設定を推奨します。

③ AI 予測（10 問）

Q26. AI 予測モデルとは何ですか？
過去のデータから将来の値（在庫消費量・売上・来客数等）を統計的・機械学習的に推定する仕組みです。勘ではなく確率分布で推定し、運用判断を統計でバックアップします。
Q27. どんな業種でも対応できますか？
予測可能なあらゆる業種に対応可能です。最初の 6 ヶ月は在庫予測・売上予測の 2 系統に集中していますが、他のユースケース（人員配置・需要予測・離脱予測等）もご相談ください。
Q28. 必要なデータの量は？
最低 6 ヶ月分（推奨 12〜24 ヶ月）の販売・出荷データが必要です。データ量が少ない場合は時系列モデル（Prophet 等）で対応できることがあります。Phase 1 で適合可否を判定します。
Q29. データのセキュリティは大丈夫ですか？
NDA 締結 + TLS 1.3 暗号化転送 + AES-256 暗号化保存 + アクセスログ保管で対応します。お預かりしたデータを外部 AI サービスに送信することはありません。
Q30. 精度はどのくらいですか？
業種・データ量・季節性で大きく異なります。Phase 1（30 万円・分析のみ）で実現可能な精度を試算したレポートを提出し、Phase 2 進行可否のご判断をいただきます。
Q31. 導入期間はどのくらい？
標準 12 週間（Phase 1 が 4 週、Phase 2 が 4 週、Phase 3 が 4 週）です。Phase ごとに進む / 止まるの判断ができます。
Q32. 既存システムと連携できますか？
API 連携・CSV 入出力・データベース直接読込み等で対応可能です。既存基幹システム（SAP / Oracle / kintone 等）との連携実績ベースで Phase 1 にて検証します。
Q33. 運用後のサポートはありますか？
Phase 3 完了後、月次のモデル再学習・精度モニタリング・改善提案を保守プラン（プレミアム枠の追加 or 別途月額契約）で対応します。
Q34. ROI はどうなりますか？
投資対効果は事業規模・業種・現状の在庫精度等で大きく異なります。Phase 1 の実現可能性レポートで個別試算します。一律の数値は提示しません（誇張防止のため）。
Q35. スモールスタートは可能ですか？
Phase 1（30 万円・2 週間・分析のみ）からのスタートが可能です。Phase 1 だけで終了するご判断もできます。

共通（5 問）

Q36. 対応エリアは？
静岡県東部（沼津・三島・富士・富士宮・裾野・長泉・清水）を主軸に、静岡県全域および全国オンライン対応可能です。AI 予測は業種を問わず全国対応です。
Q37. 支払方法は？
銀行振込（ソニー銀行 / ゆうちょ銀行）が標準です。請求書発行・領収書発行・適格請求書（インボイス）対応可能です。詳細は /pricing/ をご覧ください。
Q38. 領収書・請求書は発行されますか？
発行します。請求書は契約時または月初、領収書は入金確認後に発行します。インボイス（適格請求書）対応です。
Q39. 個人事業主でも依頼できますか？
はい。当社自身が個人事業主であり、士業・飲食・建設・不動産など個人事業主様の事例にも対応します。料金プランは規模に応じて柔軟にご提案します。
Q40. 相談だけでも可能ですか？
可能です。トップページの 1 分でできる無料診断シミュレーターで現状を可視化したあと、メールでお問い合わせください。診断結果のご相談だけでも歓迎です。

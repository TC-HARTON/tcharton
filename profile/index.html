なぜ T.C.HARTON を始めたか

静岡県東部の中小企業様から「月額制で WEB を作ったが、3 年で 100 万円払って何も育っていない」という声を何度も聞いてきました。当時、選択肢は 月額制（縛り強・所有権なし） か 買い切り（作って終わり・育たない） の二択しかなく、どちらも事業者の本当の利益にはなっていなかった。

T.C.HARTON は WEB 制作（保守込み） と AI 予測（必要時のみ） の 2 軸に切り分け、お客様が必要な時に必要な分だけ投資できる設計に組み直した個人事業です。月額保守は解約自由、所有権はお客様、未来は予測で見る。

沼津・三島・富士の事業者様の経営に、月額の重荷ではなく、育てる選択肢を。これが屋号の出発点です。

経歴と専門領域

データサイエンス歴 15 年。金融アルゴリズム開発と機械学習プロジェクトの経験を、地域中小企業の WEB と AI に応用しています。
